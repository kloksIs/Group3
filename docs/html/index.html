Система прогнозирования эпидемии

Описание

Программа моделирует распространение вируса в популяции.

Классы

  • Virus - параметры вируса (скорость заражения, летальность, продолжительность болезни)
  • VirusState - состояние активного вируса
  • Simulation - управление симуляцией

Возможности

  • Добавление/удаление вирусов
  • Просмотр статистики
  • Симуляция дней (прокрутка времени)
  • Добавление мер защиты (маски, сезон, вакцинация)

Условие победы

Эпидемия побеждена, если заражённых < 10.
